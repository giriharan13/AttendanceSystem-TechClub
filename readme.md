# Cloud-Based Attendance System using Face Recognition (Web App)

This project is a cloud-based attendance system that utilizes face recognition for authentication and tracking attendance. It is a web application built using Flask, OpenCV, liveness detection by the Silent-Face-Anti-Spoofing repository, Supabase for cloud storage and authentication, and SQLAlchemy (Flask extension for database operations).

## Features

- User registration and login functionality using Supabase authentication.
- Face detection and recognition using OpenCV and the Silent-Face-Anti-Spoofing library for liveness detection.
- Cloud-based storage and retrieval of user data using Supabase.
- Attendance tracking and reporting.

## Prerequisites

- Python 3.7 or higher
- Flask
- OpenCV
- Silent-Face-Anti-Spoofing
- Supabase
- SQLAlchemy
